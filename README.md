# Skill Tree Builder (Frontend)

This is the frontend of a full-stack Skill Tree Builder application that helps users track and improve their learning through a gamified skill progression system.

## 🚀 Tech Stack

* React.js
* JavaScript
* HTML & CSS

## 🔗 Backend Integration

The frontend is connected to a Spring Boot backend via REST APIs.

## ✨ Features

* API integration with backend
* Dynamic data fetching
* Clean UI structure
* Ready for authentication system

## 📡 API Example

The frontend fetches data from:
http://localhost:8080/test

## 🛠️ How to Run

```bash
npm install
npm start
```

App runs on:
http://localhost:3000
